**ADDON_META:**

ADDON_META = {

- If `context.get("gitops_tool") != "flux"`, return `{}`
- Otherwise, instantiate `DeploymentLifecycleGenerator` and call `generate()`

**`DeploymentLifecycleGenerator.__init__`:** Accept project_name, description, context. Extract `platform`, `gitops_tool`, `eck_enabled` (same logic as flux_deployment.py: check `sizing_context.get("eck_operator")` or `sizing_context.get("source") == "sizing_report"`).

**`generate()` returns Dict[str, str] with these 4 files:**

1. **`scripts/mirror-secrets.sh`** — Secret mirroring script:
   - Header: `#!/usr/bin/env bash`, `set -euo pipefail`, `PROJECT_NAME="{project_name}"`, `ROOT_DIR="$(cd "$(dirname "$0")/.." && pwd)"`
   - Step [1/3]: Wait for Elasticsearch to be ready: `kubectl wait elasticsearch/${PROJECT_NAME} -n ${PROJECT_NAME} --for=condition=Ready --timeout=15m`
   - Step [2/3]: Create observability namespace: `kubectl create namespace observability 2>/dev/null || true`
   - Step [3/3]: Mirror ECK elastic-user secret to observability namespace:
     ```bash
     kubectl create secret generic otel-es-credentials \
       --from-literal=username=elastic \
       --from-literal=password="$(kubectl get secret ${PROJECT_NAME}-es-elastic-user -n ${PROJECT_NAME} \
         -o go-template='{{.data.elastic | base64decode}}')" \
       -n observability --dry-run=client -o yaml | kubectl apply -f -
     ```
   - Print "Secret mirroring complete."

2. **`scripts/fleet-output.sh`** — Fleet output configuration:
   - Same header pattern
   - Step [1/4]: Wait for Kibana ready: `kubectl wait kibana/${PROJECT_NAME} -n ${PROJECT_NAME} --for=condition=Ready --timeout=5m`
   - Step [2/4]: Get ES password: `ES_PASS="$(kubectl get secret ${PROJECT_NAME}-es-elastic-user -n ${PROJECT_NAME} -o go-template='{{.data.elastic | base64decode}}')"`
   - Step [3/4]: Get CA fingerprint: `CA_FP=$(kubectl get secret ${PROJECT_NAME}-es-http-certs-public -n ${PROJECT_NAME} -o jsonpath='{.data.ca\.crt}' | base64 -d | openssl x509 -fingerprint -sha256 -noout 2>/dev/null | sed 's/.*=//;s/://g')`
   - Step [4/4]: Find Kibana pod, exec curl to PUT `https://localhost:5601/api/fleet/outputs/fleet-default-output` with JSON body containing `name: "default"`, `type: "elasticsearch"`, `hosts: ["https://${PROJECT_NAME}-es-http.${PROJECT_NAME}.svc:9200"]`, `is_default: true`, `is_default_monitoring: true`, `ca_trusted_fingerprint: "${CA_FP}"`, `config_yaml: "ssl.verification_mode: none"`
   - On success: "Fleet output configured."
   - On failure: "Fleet output config failed (may need manual setup)."

3. **`scripts/import-dashboards.sh`** — OTEL dashboard import:
   - Same header pattern
   - Step [1/3]: Get ES password (same as fleet-output.sh)
   - Step [2/3]: Find Kibana pod
   - Step [3/3]: Import dashboard via `curl -X POST "https://localhost:5601/api/saved_objects/_import?overwrite=true"` with `--form file=@/dev/stdin < "$DASHBOARD_FILE"` where `DASHBOARD_FILE="$ROOT_DIR/observability/otel-dashboards/otel-infrastructure-overview.ndjson"`
   - Handle missing dashboard file gracefully with informative message

4. **`scripts/preflight-check.sh`** — Pre-flight validation:
   - Same header pattern
   - Step [1/3]: Cluster connectivity: `kubectl cluster-info >/dev/null 2>&1` — on failure: `echo "ERROR: Cannot reach Kubernetes API. Fix: export KUBECONFIG=/path/to/kubeconfig or check cluster status with 'kubectl cluster-info'"; exit 1`
   - Step [2/3]: Flux installation: Check controllers running: `kubectl get deployment -n flux-system kustomize-controller source-controller -o name >/dev/null 2>&1` — on failure: `echo "ERROR: Flux controllers not found in flux-system namespace. Fix: flux install --namespace=flux-system"; exit 1`
   - Step [3/3]: Required CRDs: Check `kubectl get crd kustomizations.kustomize.toolkit.fluxcd.io gitrepositories.source.toolkit.fluxcd.io >/dev/null 2>&1` — on failure: `echo "ERROR: Required Flux CRDs missing (kustomizations.kustomize.toolkit.fluxcd.io, gitrepositories.source.toolkit.fluxcd.io). Fix: flux install --namespace=flux-system"; exit 1`
   - On all pass: `echo "All pre-flight checks passed."`, exit 0
