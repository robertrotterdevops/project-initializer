**Method `_generate_verify_deployment(self) -> str`:**

Generate `scripts/verify-deployment.sh` with this structure:
- Header: `#!/usr/bin/env bash`, `set -euo pipefail`, `PROJECT_NAME="{self.project_name}"`
- Define kustomization list as a bash associative array or parallel arrays mapping name to timeout in seconds:
  ```bash
  KUSTOMIZATIONS=("{project_name}" "{project_name}-infra" "{project_name}-apps")
  TIMEOUTS=(120 600 1200)
  ```
  If `self.eck_enabled`: append `"{project_name}-agents"` with timeout `1200`
- Define `POLL_INTERVAL=30`
- Print header: `echo "Deployment Verification — polling kustomization readiness"`
- Initialize results array
- For each kustomization, polling loop:
  ```bash
  ELAPSED=0
  while [ $ELAPSED -lt $TIMEOUT ]; do
    STATUS=$(kubectl get kustomization $KSNAME -n flux-system -o jsonpath='{.status.conditions[?(@.type=="Ready")].status}' 2>/dev/null || echo "Unknown")
    if [ "$STATUS" = "True" ]; then
      RESULT="Ready"
      break
    fi
    sleep $POLL_INTERVAL
    ELAPSED=$((ELAPSED + POLL_INTERVAL))
  done
  ```
  If loop exhausts timeout: `RESULT="FAILED (timeout after ${TIMEOUT}s)"`
- After all kustomizations checked, print status table:
  ```bash
  printf "%-40s %-12s %-10s %s\n" "NAME" "TIMEOUT" "ACTUAL" "STATUS"
  printf "%-40s %-12s %-10s %s\n" "----" "-------" "------" "------"
  # For each result:
  printf "%-40s %-12s %-10s %s\n" "$KSNAME" "${TIMEOUT}s" "${ELAPSED}s" "$RESULT"
  ```
- After kustomization checks, if all passed, check Elasticsearch pods:
  ```bash
  echo ""
  echo "Checking Elasticsearch StatefulSet pod health..."
  ES_PODS=$(kubectl get pods -n ${PROJECT_NAME} -l "elasticsearch.k8s.elastic.co/cluster-name=${PROJECT_NAME}" --field-selector=status.phase=Running --no-headers 2>/dev/null | wc -l)
  if [ "$ES_PODS" -gt 0 ]; then
    echo "  Elasticsearch: $ES_PODS pod(s) Running"
  else
    echo "  WARNING: No Elasticsearch pods in Running state"
    OVERALL_RESULT=1
  fi
  ```
- Exit with `$OVERALL_RESULT` (0=pass, 1=fail)

  ```bash
  echo "[1/4] Checking required directories..."

      ERRORS=$((ERRORS + 1))
    else

    fi
  done
  ```

- Step [2/4]: Check kustomization.yaml exists in each required directory:
  ```bash
  echo "[2/4] Checking kustomization.yaml files..."
  for DIR in flux-system infrastructure apps clusters/management; do
    if [ -d "$ROOT_DIR/$DIR" ] && [ ! -f "$ROOT_DIR/$DIR/kustomization.yaml" ]; then
      echo "  ERROR: Missing kustomization.yaml in $DIR"
      echo "  Fix: Add a kustomization.yaml to $DIR with appropriate resource references."
      ERRORS=$((ERRORS + 1))
    fi
  done
  ```

- Step [3/4]: Validate YAML syntax of all .yaml files:
  ```bash
  echo "[3/4] Validating YAML syntax..."
  YAML_FILES=$(find "$ROOT_DIR" -name "*.yaml" -not -path "*/\.*" -not -path "*/node_modules/*" 2>/dev/null)
  for FILE in $YAML_FILES; do
    if ! python3 -c "import yaml, sys; yaml.safe_load(open(sys.argv[1]))" "$FILE" 2>/dev/null; then
      echo "  ERROR: Invalid YAML syntax in $FILE"
      echo "  Fix: Check $FILE for YAML formatting errors (missing colons, bad indentation, tabs instead of spaces)."
      ERRORS=$((ERRORS + 1))
    fi
  done
  echo "  Checked $(echo "$YAML_FILES" | wc -w) YAML files"
  ```

- Step [4/4]: Check dangling references in kustomization.yaml files:
  ```bash
  echo "[4/4] Checking for dangling resource references..."
  KUSTOMIZATION_FILES=$(find "$ROOT_DIR" -name "kustomization.yaml" -not -path "*/\.*" 2>/dev/null)
  for KFILE in $KUSTOMIZATION_FILES; do
    KDIR=$(dirname "$KFILE")
    # Extract resource lines (skip comments, empty lines, the "resources:" key itself)
    RESOURCES=$(grep -E "^- " "$KFILE" 2>/dev/null | sed 's/^- //' | grep -v "^#" || true)
    for RES in $RESOURCES; do
      # Resource can be a file (ends in .yaml/.yml) or a directory (has kustomization.yaml inside)
      TARGET="$KDIR/$RES"
      if [[ "$RES" == *.yaml ]] || [[ "$RES" == *.yml ]]; then
        if [ ! -f "$TARGET" ]; then
          echo "  ERROR: Dangling reference in $KFILE: $RES (file not found)"
          echo "  Fix: Create $TARGET or remove the reference from $KFILE"
          ERRORS=$((ERRORS + 1))
        fi
      else
        # Directory reference — check for kustomization.yaml inside
        if [ -d "$TARGET" ]; then
          if [ ! -f "$TARGET/kustomization.yaml" ]; then
            echo "  WARNING: Directory $TARGET exists but has no kustomization.yaml"
          fi
        elif [ ! -f "$TARGET" ] && [ ! -d "$TARGET" ]; then
          echo "  ERROR: Dangling reference in $KFILE: $RES (path not found)"
          echo "  Fix: Create $TARGET or remove the reference from $KFILE"
          ERRORS=$((ERRORS + 1))
        fi
      fi
    done
  done
  ```
